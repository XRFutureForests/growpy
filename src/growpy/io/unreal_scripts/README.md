# Unreal Engine Python Scripts

This directory contains standalone Python scripts for executing GrowPy operations in Unreal Engine.

## Scripts

### import_forest.py (Template)

Template script for importing GrowPy forest USD files into Unreal Engine Content Browser.

**Configuration:**

- `USD_FILES_DIR`: Path to exported USD files
- `IMPORT_PATH`: Unreal Content Browser destination

### clean_assets.py (Template)

Template script for removing GrowPy assets from Unreal Engine Content Browser.

**Configuration:**

- `CLEANUP_PATH`: Unreal Content Browser path to clean
- `DRY_RUN`: Set to `True` for preview mode

### Auto-Generated Scripts

When you run GrowPy CLI commands with `--import-to-unreal`, customized scripts are generated:

- `import_forest.py` (in output directory's unreal_scripts/)
- `clean_assets_generated.py` (generated by clean_unreal_assets.py)

## Usage

### Method 1: VSCode Unreal Python Extension (Recommended)

1. Install VSCode extension: "Unreal Python" by Nils Soderman
2. Open script in VSCode
3. Right-click file > **"Execute Python File in Unreal"**
4. Watch Output Log in Unreal for results

### Method 2: Unreal Python Console

1. Open Unreal Editor
2. Window > Developer Tools > Output Log (enable "Python" category)
3. Window > Developer Tools > Python Console
4. Execute:

   ```python
   exec(open(r'C:\path\to\script.py').read())
   ```

### Method 3: Command Line

```bash
UnrealEditor-Cmd.exe "C:\path\to\project.uproject" -run=pythonscript -script="C:\path\to\script.py"
```

## Requirements

### Unreal Engine Plugins

Must be enabled in Edit > Plugins:

- **Python Editor Script Plugin** (built-in)
- **Editor Scripting Utilities** (built-in)
- **USD Importer** (for import_forest.py)

### VSCode Extension (Optional but Recommended)

- **Unreal Python** by Nils Soderman
- Provides right-click execution in VSCode
- Real-time feedback in Output Log

## Workflow Example

```bash
# 1. Generate forest with Unreal import script
conda activate the-grove
python src/growpy/cli/generate_forest.py --quality high --import-to-unreal

# 2. Open generated script in VSCode
# Location: data/output/forest/unreal_scripts/import_forest.py

# 3. Right-click > "Execute Python File in Unreal"

# 4. Check Unreal Content Browser at /Game/GrowPy/Trees

# 5. (Optional) Generate cleanup script
python src/growpy/cli/clean_unreal_assets.py --dry-run

# 6. Execute cleanup script to preview or delete
```

## Troubleshooting

### Import Errors

- Verify USD Importer plugin is enabled
- Check USD file paths are correct (use absolute paths)
- Ensure Editor Scripting Utilities plugin is enabled

### Script Not Found

- Use absolute paths (backslashes escaped: `r'C:\path\to\file.py'`)
- Or forward slashes: `'C:/path/to/file.py'`

### No Output

- Enable "Python" category in Output Log
- Check Python plugin is enabled: Edit > Project Settings > Plugins > Python

## Script Templates vs Generated Scripts

**Templates** (`import_forest.py`, `clean_assets.py`):

- Generic scripts with placeholders
- Edit configuration section before running
- Reusable for different projects

**Generated Scripts**:

- Customized for specific GrowPy exports
- Pre-configured with correct paths
- Ready to execute without editing
