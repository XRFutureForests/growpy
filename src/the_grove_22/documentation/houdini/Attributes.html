Attributes

The Grove Build node adds several attributes to the geometry’s points and primitives. They start with the gr_ prefix.


Work in progress - most of the attributes are self explanatory, but for an explanation on each of the attributes, please look at the documentation of the_grove_core.Model. The Houdini add-on is a work in progress - if you need specific attributes to integrate The Grove in your pipeline, let me know.


Miscellaneous

While The Grove in Blender builds each tree as a separate object, The Grove in Houdini builds all trees in a single geometry node. You can use the primitive attribute gr_tree_id to single out a tree.



Similarly, you can add a Blast node with @gr_dead==1 to remove dead branches.
