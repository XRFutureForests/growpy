Introduction

The Grove in Houdini is brings the fun of growing trees into the powerful node system of Houdini. The Grove in Houdini is a set of SOP geometry nodes, that let you build a chronologic sequence of events like planting trees, simulating years of growth, pruning, changing growth parameters, and then growing several more years with these changes.


With each step, the evolving simulation is passed along from node to node as a detail attribute on the geometry. Then at the end, the Build node creates the actual tree geometry. Or use the Skeleton node to build a dynamics skeleton. At that point, the simulation data is no longer needed, and you can safely delete the grove attribute. The Build node does still include the simulation data so that you can create a growth animation within a Solver node.


The Build node also adds a handy set of attributes to the tree geometry, which you can use to further customize the output geometry and materials, add animation, and duplicate twigs.


The nodes

The example file TheGroveExample.hip shows you how you can link the nodes together.


Grove Plant

You start with a Plant node that creates a number of placeholders where your trees will start to grow. The Plant node creates all kinds of natural and cultural arrangements. The node creates a set of poly lines to visualize the position and direction of growth for the planted trees.


Grove Properties

Then use a Grove Properties node to set all the growth parameters that define the character of a tree species. There are many parameters that all depend on each other, so you’ll want to start by loading one of the many presets. You can find the presets in the_grove_22/presets.


Grove Simulate

The simulate node grows the trees for a number of flushes.


Grove Prune

Prune the trees with geometry. Branches are cut at the point of intersection with the geometry, and the remainder of the branch is removed.


This is a work-in-progress node. A interactive pruning experience like the one in Blender is being worked on.



Grove Smooth

This optional node will smooth any rough kinks in the tree’s branches. You can use it just before building the tree model.


Grove Build

Create the final tree models as polygonal geometry with a range of attributes. You can control the Resolution at the base of the tree, and then Reduce that number when the branches get thinner. Most of the tree’s geometry is in the thousands of thin branch ends, which can do with a lot less polygons without a noticeable drop in quality. Use UV Repeat to repeat the bark texture around the girth of the tree, to automatically diminish on thinner branches.


The built model adds several attributes, which start with the gr_ prefix.


Grove Import

Load a previous growth simulation from a .grove file. You can continue simulating extra flushes of growth or simply link the output to a Build node.


You can also use the Grove Import node to transfer trees to-and-from the Blender add-on. The Blender side of The Grove is a completely different user experience, both Houdini and Blender have their strengths, and the Blender add-on is great for quick growing, experimentation and artistic control.


Grove Export

Save the simulation to a .grove file. Use this as a cache, or to save a cool tree for later. Or use it to bring in trees grown in the Blender add-on.
