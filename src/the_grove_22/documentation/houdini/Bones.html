Bones

See also: Skeleton


The Bones node creates an extremely optimized skeleton with way less bones. It’s great for adding wind sway or animating the tree manually.


Parameters

  • Reduce - increasingly skip adding bones to thinner branches.
  • Bias - create more bones either up high in the crown or down at the base of the tree.
  • Length - increase the maximum length of each bone to significantly reduce the number of bones.

Note that The Grove in Blender has the option to create disconnected bones with the Connected option. This option is not exposed in Houdini, because Houdini expects bones to always be connected.


Usage

Add a Grove Bones node after the simulation, and right before building the model. Then branch off to both a Grove Build node and a Skeleton node. This way the bone weights in the build will match the bones in the skeleton. Here’s a screenshot of the_grove_example.hip that shows the wiring. This file will also help you with setting the bone weights and the wind sway animation node is a good inspiration to build upon.
