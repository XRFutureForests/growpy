# [the_grove_core](./the_grove_core.html).about

The sub module `the_grove_core.about` contains some informative strings about The Grove.
  * about (string) - A short description of the program.
  * license (string) - The EULA for The Grove.
  * release (string) - The release of The Grove, for example `2.2`.
  * edition (string) - The edition of The Grove, either `starter`, `indie` or `studio`.
